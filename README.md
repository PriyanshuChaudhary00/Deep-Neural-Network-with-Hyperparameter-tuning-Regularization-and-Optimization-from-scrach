🧠 Deep Neural Network from Scratch with Hyperparameter Tuning, Regularization & Optimization

A complete implementation of a Deep Neural Network from scratch using only NumPy, without relying on deep learning frameworks such as TensorFlow or PyTorch.

The objective of this project was to understand the mathematics and implementation details behind neural networks by building every component manually.

⸻

🚀 Features

* ✅ Deep Neural Network built from scratch
* ✅ Forward Propagation
* ✅ Backward Propagation
* ✅ He Initialization
* ✅ Mini-batch Gradient Descent
* ✅ L2 Regularization
* ✅ Momentum Optimizer
* ✅ Adam Optimizer
* ✅ Hyperparameter Tuning
* ✅ Binary Classification
* ✅ Modular and reusable implementation
